# QR-Code-Generator-Project
This project uses the qrcode Python library to generate a customizable QR code. Specifically, it takes a GitHub repository URL, creates a QR code image with a purple fill color, and saves it as 'Preetika_Github.png'. The generated image can then be downloaded to your local machine.
